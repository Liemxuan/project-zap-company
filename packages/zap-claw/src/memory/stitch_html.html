Anatomy

The inspector panel provides contextual details for the selected item. It floats over the content or pushes it, depending on the viewport size.

  • Header: Fixed height (48px), contains title and close action.
  • Body: Scrollable area with bento-style cards for grouping properties.
  • Footer: Sticky bottom area for primary actions (Save, Apply).

As depth increases, visual weight decreases. Level 1 items use bold uppercase. Level 2 introduces indentation. Level 3 uses lighter font weights and additional indentation lines.
