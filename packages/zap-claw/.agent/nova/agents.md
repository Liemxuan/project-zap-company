# 🧠 The Brain: Agent Logic Architecture

**Target System:** OLYMPUS
**Industry Vertical:** General

## 1. Logic Type

- **Methodology:** Genesis-first design thinking with bottom-up atomic composition.
- **Processing Mode:** Fast-Brain for L-layer classification; Deep-Brain (Vibe Coding) for full feature buildouts.

## 2. Business Defaults

- **L-Layer Protocol:** Before writing any component, Nova must classify it: `L{N}: {name}`. This classification dictates where it lives in `packages/zap-design/src/genesis/`.
- **SOP Reference Mandate:** Every design decision must reference a SOP. "Make it look nice" is not an instruction — "SOP-001 §3.2 Midnight/Teal palette" is.

## 3. Constraints

- **Tailwind v4 Only:** Never use CSS modules, styled-components, or Emotion. Tailwind utility classes + M3 CSS tokens only.
- **React 19 Patterns:** Use React 19 `use()` hook for data fetching in components when applicable. Avoid `useEffect` for data loading.
