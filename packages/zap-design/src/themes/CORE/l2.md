# L2 Cover Data Dictionary (Core Theme)

> [!IMPORTANT]
> **ZAP-AUDIT DIRECTIVE:** This file is the absolute source of truth for the **L2 Cover Layer**. The `zap-audit` skill must use this to verify the primary application sandbox.

## 1. Definition & Role
The **L2 Cover** (The Sandbox) is the structural holding tank for the actual user interface. It sits on top of the L1 Canvas and defines the core content boundaries (e.g., a massive central white rectangle over a gray L1 canvas).

## 2. Structural Tokens
- **Mandatory Background:** `bg-layer-cover`
- **Dynamic Borders:** `style={{ borderRadius: 'var(--layer-2-border-radius)' }}` (Inherits L1 settings from the Inspector/Design Tokens).
- **Shadow/Depth:** May utilize global L2 drop shadows if configured by the theme layout.

## 3. Topographic Rules
1. **Padding Contract:** L2 MUST contain internal padding to protect its children (usually `p-6` or `p-8`, or mapped to `var(--layer-2-padding)`).
2. **Margin Constraint:** It must NOT bleed directly into the window edges. It needs margins or flex-gaps separating it from the L1 Canvas viewport edges.
3. **No Direct Primates:** Do not put raw text nodes directly inside L2. Form elements, cards, and data tables belong in **L3 Panels** (inside the L2 Cover).

## 4. Mandatory Audit Grep Scans
The `zap-audit` process MUST flag any violation of the L2 structural contract.

```bash
# Sweep for L2 covers that have hardcoded inline backgrounds overriding the token
grep -rE 'class(Name)?="[^"]*bg-layer-cover[^"]*"' src/ --include="*.tsx" | grep -E 'style=\{\{.*background.*\}\}'

# Sweep for L2 covers that hardcode structural radiuses instead of inheriting the L2 dynamic variable
grep -rE 'class(Name)?="[^"]*bg-layer-cover[^"]*rounded-' src/ --include="*.tsx"
```

## 5. Resolution Guide for `zap-fix`
If an L2 layer fails the audit:
1. Strip hardcoded border radiuses (`rounded-xl`) and map them to `style={{ borderRadius: 'var(--layer-2-border-radius)' }}`.
2. Strip explicit `bg-white` classes, enforcing `bg-layer-cover`.
3. Ensure L2 is structurally distinct from L1 (it cannot possess `min-h-screen` if it shares the same node as `bg-layer-canvas`).
